Introduction

Welcome to the User Guide for the Tab Manager extension! This guide will walk you through the features and usage of the extension to help you manage your browser tabs efficiently.

Installation

  1. Open the Chrome Web Store.
  2. Search for "Tab Manager".
  3. Click on the "Add to Chrome" button next to the Tab Manager extension.
  4. Follow the prompts to install the extension.
  5. Once installed, you'll see the Tab Manager icon in the Chrome toolbar.

Using the Extension

Opening the Popup

To open the Tab Manager popup, follow these steps:

  1. Click on the Tab Manager icon in the Chrome toolbar.
  2. The popup window will appear, displaying a list of your open tabs.
  3. or, press ctrl+shift+e on windows
  4. and press cmd/ctrl+shift+e on mac

Viewing Tabs

In the Tab Manager popup, you can view a list of all your open tabs. Each tab is represented by its title and favicon (if available).

Closing Tabs

To close a tab, follow these steps:

  1. In the Tab Manager popup, hover over the tab you want to close.
  2. Click on the 'X' button next to the tab.
  3. The tab will be closed immediately.

Switching Tabs

To switch to a different tab, follow these steps:

  1. In the Tab Manager popup, click on the tab you want to switch to.
  2. The selected tab will become active, and its window will be brought to the front.

User Settings

The Tab Manager extension currently does not offer user settings. All features are available directly from the popup interface.
